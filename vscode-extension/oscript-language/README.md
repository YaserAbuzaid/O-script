# O-script Language Support (local)

This is a tiny VS Code extension that adds:

- Syntax highlighting for **.os** files (O-script)
- Useful snippets (`class`, `fun`, `if`, `while`, `checkpoint`, ...)

## Install (no account / no marketplace)

### Method A (recommended): Copy into your VS Code extensions folder

1. Find your VS Code extensions folder:
   - **Windows**: `%USERPROFILE%\.vscode\extensions`
   - **macOS**: `~/.vscode/extensions`
   - **Linux**: `~/.vscode/extensions`

2. Copy this folder **oscript-language** into that extensions folder.

3. Restart VS Code.

### Method B: Install from a local folder

1. In VS Code press `Ctrl+Shift+P` (or `Cmd+Shift+P` on macOS)
2. Run: **Developer: Install Extension from Location...**
3. Select the **oscript-language** folder
4. Restart VS Code

After installation, `.os` files should highlight keywords like `class`, `fun`, `var`, etc.
